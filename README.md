# 🚰 Eigen Variance Index (EVI) for Pipeline Leak Detection

**Unsupervised vibration-based pipeline leak detection using PVDF sensors and Eigen Variance Index (EVI) for detection, localisation, and quantification under varying operating conditions.**

---

## 📌 Overview

Leakage in pressurised pipelines often produces subtle vibration signatures that are difficult to distinguish under varying operational conditions. This project presents a **physics-informed, data-driven framework** that leverages **flow-induced vibration signals** measured using flexible PVDF sensors.

The core contribution is the introduction of the **Eigen Variance Index (EVI)** — a covariance-based indicator derived from multichannel sensor data that enables:

- Reliable **leak detection**
- Spatial **localisation**
- Quantification of **leak severity**

Unlike conventional methods, the framework operates in an **unsupervised setting** and remains robust under **noise and flow variability**.

---

## 🔬 Key Contributions

- 📈 **Eigen Variance Index (EVI)** for damage detection
- 🔍 Unsupervised framework (no labelled damage data required)
- 📍 Sensor-wise comparison for **leak localisation**
- 📊 Quadratic relationship between EVI and leak size (**R² > 0.98**)
- 🔊 Robust performance under **low signal-to-noise ratios**
- ⚙️ Experimental validation across multiple flow conditions

---

## ⚙️ Methodology

### 1. Sensing Mechanism
Flexible **PVDF piezoelectric sensors** are surface-mounted on the pipeline to capture vibration-induced voltage signals generated by flow–structure interaction.

### 2. Multivariate Representation
The multichannel response is assembled into a data matrix:

X ∈ ℝ^(N × M)

where:
- N = number of time samples  
- M = number of sensors  

The covariance matrix is computed as:

C = (1 / (N - 1)) XᵀX

### 3. Eigen Variance Index (EVI)

The dominant eigenvalue of the covariance matrix is defined as:

EVI = λ₁

- Represents the **maximum variance direction**
- Acts as a proxy for **disturbance energy**
- Increases **monotonically with leak diameter**

---

## 🧪 Experimental Setup

- Closed-loop water pipeline system  
- Steel test section with controlled leak diameters: **2–10 mm**  
- Flow rates: **7–24 GPM**  
- 3 PVDF sensors placed along pipe length  
- Sampling frequency: **2 kHz**  
- 150 datasets per sensor for statistical robustness  

---

## 📈 Results

### ✅ Damage Detection
- Clear separation between **healthy and damaged states**
- Thresholds derived from **baseline statistics**
- Consistent performance across all flow conditions  

### 📍 Damage Localisation
- Sensor closest to leak shows **maximum EVI deviation**
- Enables reliable **relative localisation within a segment**

### 📊 Damage Quantification
- Strong quadratic relationship between EVI and leak diameter  
- Enables approximate **severity estimation**

### 🔊 Noise Robustness
- Maintains **>95% correlation** under noisy conditions  
- More stable than traditional **RMS-based indicators**

---

## 🧠 Why This Work Matters

Conventional pipeline monitoring approaches often:
- Rely on **absolute thresholds**
- Struggle under **operational variability**
- Require **labelled datasets**

This framework overcomes these limitations by:
- Using **regime-conditioned baselines**
- Focusing on **variance redistribution** instead of amplitude
- Operating in a fully **unsupervised manner**

---

## ⚠️ Limitations

- Designed for **segment-level monitoring**, not full network-scale localisation  
- Requires **baseline data under known healthy conditions**  
- Performance may vary with **soil interaction and pipeline complexity**
